# 编译

```shell
mvn -T 2C compile -DskipTests -Dmaven.javadoc.skip -Dfindbugs.skip -Dcheckstyle.skip \
  -Dlicense.skip -pl '!webui'
```





# 启动

```shell
# step 1
alluxio format
# step 2
alluxio-start.sh local SudoMount

```



# 挂载

```shell
alluxio fs mount alluxio://localhost:19998/mnt/hdfs hdfs://hadoop-master1:9000/
```
